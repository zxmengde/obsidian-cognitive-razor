# 常用命令（PowerShell）
- 安装依赖：`npm install`
- 开发构建（watch）：`npm run dev`
- 生产构建：`npm run build`
- 单元/集成测试：`npm test`
- 测试 watch：`npm run test:watch`
- 测试 UI：`npm run test:ui`
- 覆盖率：`npm run test:coverage`
- ESLint 检查：`npx eslint main.ts` 或 `npx eslint ./src/`
- 版本号提升脚本：`npm run version`
- 发布前检查：`npm run prepare-release`（构建 + 验证发布文件）
- 查看/编辑主要文档：`docs/公理化设计文档.md`, `AGENTS.md`, `docs/快速开始指南.md` 等。

# 项目概览
- 名称：Obsidian Cognitive Razor（公理化知识管理插件）
- 目标：将模糊想法标准化为结构化概念，支持语义去重、AI 内容生成、增量改进、可逆写入和任务队列管理。
- 架构：分层单向依赖 UI → 应用(core) → 数据(data)，入口 main.ts，bundle 成 main.js 供 Obsidian 加载。
- 主要目录：
  - src/core/* 应用层：队列、锁、去重、提示词、Provider、向量索引、撤销、管线编排等。
  - src/data/* 数据层：文件存储、日志、设置存储、验证器。
  - src/ui/* UI：工作台/队列/撤销视图、命令分发、设置面板、模态框、状态栏徽章等。
  - prompts/ 提示词模板；docs/ 设计与指南；data/ 运行时数据（vector-index、queue-state、duplicate-pairs、snapshots 等）。
- 关键设计依据：docs/公理化设计文档.md；AGENTS.md（Obsidian 插件开发规范）。
- 运行环境：Obsidian ≥1.5.7，Node.js 22+（开发），TypeScript 5.7+，esbuild 0.25.x。
- 发布工件：main.js, manifest.json, 可选 styles.css；不应提交 node_modules 或生成文件。
